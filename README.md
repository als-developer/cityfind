
## Overview
CITYFIND is a comprehensive e-commerce platform with:
- **Distance-Based Search** - Products sorted by proximity to user
- **Buyer Dashboard** - Shop and browse products
- **Seller Dashboard** - Manage products and orders
- **Admin Dashboard** - Full platform control (hidden)

## Features
- 📍 **Distance-Based Search** - Products sorted closest first
- 🔐 Secure authentication system
- 🛒 Shopping cart functionality
- 📱 Mobile-responsive design
- 🤖 AI-powered chatbot
- 📦 Product management
- 📊 Analytics and reporting
- 💳 Payment processing
- 🚚 Delivery tracking
- 📷 QR code scanning
